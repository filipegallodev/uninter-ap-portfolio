  1. Decodificador

    Projeto desenvolvido dentro do programa Oracle Next Education, uma parceria entre a Oracle e Alura.
    Ele consiste em criptografar qualquer texto inserido e depois realizar a descriptografia, utilizando as chaves definidas pelo programa.

  2. Lista de Filmes

    Este foi um projeto pessoal simples para elaborar uma lista pessoal de filmes para assistir e de filmes já assistidos.
    Nele possui os filmes separados por categorias e dentro de cada uma possui uma lista com nome, descrição, duração e trailer de cada filme.
